##Question:

Using the following:
 
```java
public static Func<int, int> Y(Func<Func<int, int>, Func<int, int>> f)
{
	return x => f(Y(f))(x);
}
```

Create a Func<int, int> which computes the n-th fibonacci number using only anonymous lambdas.
